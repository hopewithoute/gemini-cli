# Gemini CLI

## 🎉 Penambahan Terbaru (What's New)

Berikut adalah fitur dan pembaruan terbaru pada Gemini CLI:

- **Pengalih Multi-Akun (Multi-Account Switcher):** Kelola dan beralih antara
  beberapa profil API Gemini dengan mudah. Fitur ini memudahkan Anda yang
  memiliki lebih dari satu akun atau environment untuk berpindah profil tanpa
  hambatan.
  [Baca dokumentasinya](./docs/plans/2026-03-02-multi-account-switcher.md)

- **Baris Status Kustom & Tampilan Kuota (Custom Status Line & Quota Display):**
  Personalisasikan baris status CLI Anda dan pantau penggunaan kuota secara
  real-time. Ini sangat berguna untuk memastikan batas penggunaan API Anda tidak
  terlampaui. [Baca dokumentasinya](./docs/cli/custom-statusline.md)

- **Sistem Meta-Tools:** Kemampuan pencarian dan eksekusi berbasis agen tingkat
  lanjut. Sistem ini memungkinkan Gemini CLI beroperasi secara lebih cerdas
  untuk menemukan solusi dan menjalankan perintah kompleks dalam alur kerja
  Anda. [Baca dokumentasinya](./docs/plans/2026-03-03-meta-tools-design.md)
